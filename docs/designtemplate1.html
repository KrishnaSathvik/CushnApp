Minimal Product Prompt 
You are given an existing landing page layout.

            Do NOT change:
            - Section order
            - Content meaning
            - Information hierarchy

            Redesign this landing page using a Minimal Product design style.

            Design intent:
            The design should feel calm, refined, and intentionally restrained. Every element should feel necessary. Avoid visual noise and unnecessary decoration.

            Visual and layout rules:
            - Use generous whitespace and clear vertical rhythm
            - Prefer a simple, centered or balanced grid
            - Use a neutral color palette (white, off-white, light gray, subtle accent)
            - Keep borders subtle or remove them entirely
            - Avoid visual clutter or excessive components

            Typography:
            - Use a modern, clean sans-serif font
            - Clear hierarchy through size and weight, not color
            - Headlines should feel confident but not aggressive

            Imagery:
            - Use imagery sparingly
            - Images should feel intentional and well-aligned
            - Avoid overlays, heavy effects, or dramatic crops

            Interaction and motion:
            - Subtle, smooth transitions only
            - Motion should feel calm and almost invisible

            Overall outcome:
            The final design should feel timeless, focused, and product-led, with clarity and simplicity as the primary strengths.
Tip: you can edit this prompt after copying.
